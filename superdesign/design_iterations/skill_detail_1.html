Description

Full read/write access to Jira for AI agents. Query issues, create tickets, update statuses, manage sprints, and generate reports. Supports Jira Cloud and Jira Data Center.

Capabilities

  • Query issues with JQL

    Use any JQL query to find and filter Jira issues

  • Create and update tickets

    Full CRUD operations on issues, subtasks, and epics

  • Manage sprint boards

    Move issues between sprints, start/complete sprints

  • Generate burndown reports

    Automated sprint velocity and burndown data extraction

  • Comment on issues

    Add, edit, and read comments with @mentions
